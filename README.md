## Axure10的部分变化

### 		**组件（Componnets）代替模板（Master）**

​			**动态面板：**

```
主要用途就是实现一些动态的交互效果。所以，如果动态面板使用不熟悉的话，对axure原型的制作会有很大的影响。
作用：
  	1、  隐藏与显示

	2、  滑动效果

	3、  拖动效果

	4、  多状态效果
```

界面交互：

事件 动作 情况
